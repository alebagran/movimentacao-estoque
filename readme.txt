Para rodar na máquina de vocês:
Extrair o ZIP
Entrar na pasta e executar npm install
Vejam os comandos no LEIAME.txt
Lembrando que, para rodar a página web, é preciso levantar um servidor HTTP na pasta raiz do projeto. Exemplo: php -S localhost:80 e abrir essa URL no browser. Como o armazenamento está sendo feito no localStorage não deve haver nenhum produto cadastrado lá (a não ser que vocês criem manualmente).
